你现在拥有一个运行完美的 **AI 音乐专辑生成器**，这不仅是一个工具，更是你最好的**活体教材**。本系列文档将拆解这个项目，告诉你每一行代码背后的逻辑，教你如何从"复制粘贴"进化到"独立创造"。

## 🚀 学习路径概览

我们将学习过程分为四个阶段，对应项目的四个核心部分：

| 阶段 | 核心技能 | 对应项目文件/目录 | 目标 |
| :--- | :--- | :--- | :--- |
| **1. 前端觉醒** | React, HTML/CSS, JS | `frontend/src/App.jsx`, `components/*.jsx` | 能画出界面，能让按钮有反应 |
| **2. 后端逻辑** | Java, Spring Boot | `backend/.../controller/*`, `service/*` | 能写出 API，能处理数据逻辑 |
| **3. AI 魔法** | HTTP API, JSON | `QwenLLMService.java`, `StableDiffusionService.java` | 能指挥 AI 帮你干活 |
| **4. 架构统筹** | 部署, 调试, 全栈思维 | `application.yml`, `package.json`, 报错日志 | 能独立解决 Bug，把程序跑起来 |

## 📚 推荐学习方法 (The "Reverse Engineering" Way)

传统的学习是看书 -> 写代码。
**高效的学习是：看运行中的代码 -> 修改它看看会发生什么 -> 查资料理解为什么。**

### 你的专属练习题：

1.  **改字**：去 `frontend/src/App.jsx` 把标题从 "AI 音乐专辑生成器" 改成 "我的超级音乐工坊"，看浏览器怎么变。
2.  **改色**：去 `frontend/src/index.css` 把背景色换掉。
3.  **加Log**：去 `backend/.../AIController.java` 里加一句 `System.out.println("我被调用啦！");`，然后上传音乐，看黑框框里的输出。
