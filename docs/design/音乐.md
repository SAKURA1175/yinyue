1. 📘 项目概述

目标是构建一个能让用户：

上传音频 → 自动识别歌曲信息

上传图片 → 用于辅助生成封面

输入网易云链接 → 自动解析并展示歌曲信息

使用 AI 分析音乐 → 自动生成视觉风格、封面Prompt

自动生成精美的专辑封面图

提供一个类网易云的全局播放器

允许用户保存或分享生成专辑


该网站类似“AI 专辑封面制作器 + 音乐解析器 + 播放器”。

3. 🎨 前端 React 方案

使用 React + Vite 或 Next.js + TailwindCSS + shadcn/ui + Zustand。


---

3.1 页面结构规划

① 首页 /Home

上传音频组件

输入网易云链接

最近生成的封面展示

全局播放器置底


② 上传页面 /upload

展示上传的文件信息 + 解析进度。

③ AI 解析页面 /analyze

左右布局：

左：歌曲信息

中：AI 解析结果（风格 / 氛围 / 封面建议）

右：可编辑 Prompt


④ 封面生成页面 /generate

展示 AI 图结果（网格）

再生成按钮

修改 Prompt

下载 / 保存


⑤ 播放器 Player

底部固定

播放进度条

播放队列

点击可进入全屏



---

3.2 前端关键技术

功能	技术

文件上传	React + File Input + Axios
全局缓存	Zustand
UI 组件	Shadcn/ui
预览播放	HTML Audio API
AI 解析展示	Markdown + JSON Viewer

4.2 Spring AI 使用方式

AI 解析模块使用 LLM

ChatResponse response = chatModel.call(Prompt.of(promptString));

Prompt 模板

resources/prompts/music-analysis.st

根据以下音乐信息生成专辑封面设计建议：
标题：{title}
歌词：
{lyrics}

输出要求：JSON格式，包括
- theme
- mood
- visual_style
- colors
- image_prompt_en


---

4.3 音频识别模块

流程：

1. 调用 Audd API 识别 → 如果返回成功则直接使用


2. 否则调用 Qwen ASR → 得到歌词文本 → 交给 LLM 解析




---

4.4 网易云解析模块

流程：

1. 从链接获取音乐 ID


2. 调用第三方 NetEaseCloudMusic API


3. 提取：

歌名

歌手

专辑封面 URL

歌词（如果有）



4. 送入 AI 解析模块




---

4.5 生图模块（Image Generation）

使用 Flux，SDXL，或通义万相

输入 Prompt + 上传图片作为参考图

输出封面图（1:1）
